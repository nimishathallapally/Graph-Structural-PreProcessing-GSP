% Abstract
\abstract

Modern recommendation systems are just black boxes, they lack in transparency and computational scalability for large graphs. This research proposes a framework designed to improve both efficiency and trustworthiness in graph-based recommendation systems. To address the computational bottleneck in modern graph based recommendation systems,An intrinsic graph compression framework is developed that exploits structural properties of graphs —such as Forman-Ricci Curvature and Spectral Conductance—to identify structure redundancy in user item graphs and prune them

To ensure trustworthiness, the framework incorporates explanation module. This module provides explanations by identifying the minimal set of interactions responsible for a recommendation. By combining structuring compression with  counterfactual reasoning, this work contributes to the development of scalable, human-centric AI systems aligned with the principles of responsible AI for social empowerment in sensitive domains.

\vspace{10pt}
\noindent \textbf{Keywords:} Graph Neural Networks; Forman-Ricci Curvature; Counterfactual Explanations; Spectral Conductance; Recommender Systems.
